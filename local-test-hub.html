EE98 — local testing

Open this file from your file manager (like your EE98-Race folder), not Cursor’s side preview. Use two real browsers: e.g. Chromium/Chrome for the instructor screen and Firefox for the student (right-click winner.html → Open with → Firefox).

Instructor — leaderboard Class portal (optional check) Student — winner (open in Firefox)

If pages act blank or Firebase errors

Some setups need HTTP instead of file://. In a terminal in this folder run:

python3 -m http.server 8765 --bind 127.0.0.1

Then use: http://127.0.0.1:8765/leaderboard.html (and same host for portal / winner).
